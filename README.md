# 💍 VowGlow – AI Wedding Beauty OS

An AI-powered premium bridal beauty planning and booking platform built for the AI Startup Buildathon 2026.

VowGlow helps brides discover elite beauty professionals, generate personalized beauty plans, manage secure escrow bookings, and organize their complete wedding beauty journey inside an elegant digital workspace.

Traditional salon marketplaces focus only on basic appointments.

VowGlow extends the experience through interactive styling sandboxes, AI-assisted planning, personalized beauty timelines, real-time consultation messaging, and secure wax-sealed escrow bookings.

# 🚀 Features

### 👰 Bride Suite Experience
* **AI Bridal Planner**: Instantly designs structured treatment count-downs and budget estimates aligned with the wedding date.
* **Atelier Discovery**: Explore vetted beauty professionals with dynamic interactive filters (hair styling, draped cosmetics, makeup, airbrush).
* **Cosmic Experience Room**: A visual beauty sandbox playground to test and preview hair hues, cosmetics, luxury draping, and customized accessories.
* **Secure Checkout**: Features a physical wedding-card style statement, itemized breakups, central/state taxes, and a secure wax-sealed stamp.
* **Bridal Dashboard**: Keeps track of wedding countdowns, checklists, secure escrow booking status, and chat inboxes.

### 💅 Salon/Vendor Command
* **Atelier Command Portal**: Manage incoming bridal inquiries, trial sessions, and wedding-day calendar slots.
* **Status Controls**: Confirm, complete, or reject escrow bookings with direct updates logged on the live-feed notification system.

### 🌟 Royal Aesthetics
* **Gilded Canvas Physics**: Interactive, GPU-accelerated gold-leaf and gold-foil flakes that tumble, flutter, and spray dynamically upon hover or click/tap gestures.
* **Wedding Card Motif**: Built on rich ivory cardstock linen pattern backgrounds, double-gilded dashed borders, and elegant Cormorant Garamond display typography.

# 📦 Installation & Local Run

1. **Clone the repository:**
   ```bash
   git clone [repository-url]
   cd vowglow
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure environment secrets:**
   Create a `.env` file in the root directory:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Launch development server:**
   ```bash
   npm run dev
   ```

5. **Build for production deployment:**
   ```bash
   npm run build
   ```

6. **Start production server:**
   ```bash
   npm run start
   ```
